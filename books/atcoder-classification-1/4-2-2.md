```python:サンプルコード：sample_228.py
for i in range(5):
    print(i)
```

```text:実行結果
0
1
2
3
4
```

```python:サンプルコード：sample_229.py
for i in range(2, 5):
    print(i)
```

```text:実行結果
2
3
4
```

```python:サンプルコード：sample_230.py
for i in range(2, 5, 2):
    print(i)
```

```text:実行結果
2
4
```

```python:サンプルコード：sample_231.py
for i in range(3, -1, -1):
    print(i)
```

```text:実行結果
3
2
1
0
```
