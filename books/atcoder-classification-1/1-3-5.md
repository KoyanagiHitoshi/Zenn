title: "　　1.3.5 区切り文字の指定"
---

```text:入力
a,b,c,d
```

```python:サンプルコード
x = list(input().split(","))
print(x)
```

```text:実行結果
['a', 'b', 'c', 'd']
```

```text:入力
2019/02/26
```

```python:サンプルコード
y, m, d = map(int, input().split("/"))
print(y, m, d)
```

```text:実行結果
2019 2 26
```

```text:入力
10+20+30
```

```python:サンプルコード
x = list(map(int, input().split("+")))
print(sum(x))
```

