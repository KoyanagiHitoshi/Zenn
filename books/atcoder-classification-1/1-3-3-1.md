整数を複数の変数に格納する。

```text:入力
1
2
```

```python:サンプルコード
x = int(input())
y = int(input())
print(x, y)
```

```text:実行結果
1 2
```

内包表記を利用して、整数を複数の変数に格納する。

```text:入力
1
2
```

```python:サンプルコード
x, y = [int(input()) for i in range(2)]
print(x, y)
```

```text:実行結果
1 2
```

整数をリストに格納する。

```text:入力
1
2
3
```

```python:サンプルコード
x = [int(input()) for i in range(3)]
print(x)
```

```text:実行結果
[1, 2, 3]
```

入力行数（N = 3）が指定され、整数をリストに格納する。

```text:入力
3
1
2
3
```

```python:サンプルコード
N = int(input())
x = [int(input()) for i in range(N)]
print(x)
```

```text:実行結果
[1, 2, 3]
```
