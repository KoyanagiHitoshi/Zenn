* [数値型](https://docs.python.org/ja/3/library/stdtypes.html#numeric-types-int-float-complex)

浮動小数点数を出力する。

```python:サンプルコード：sample_3.py
print(2.9)
```

```text:実行結果
2.9
```

浮動小数を変数に格納して出力する。

```python:サンプルコード：sample_4.py
x = 3.5
print(x)
```

```text:実行結果
3.5
```
