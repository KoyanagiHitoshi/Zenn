```python:サンプルコード
x = [1, 2, 3]
for i in x:
    print(i)
```

```text:実行結果
1
2
3
```

```python:サンプルコード
x = ["a", "b", "c"]
for i in x:
    print(i)
```

```text:実行結果
a
b
c
```

```python:サンプルコード
X = ["a", "b", "c"]
Y = [10, 20, 30]
for x, y in zip(X, Y):
    print(x, y)
```

```text:実行結果
a 10
b 20
c 30
```
