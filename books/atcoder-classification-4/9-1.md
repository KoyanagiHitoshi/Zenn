```python:サンプルコード
stack = [3, 4, 5]
stack.append(6)
print(stack)
stack.pop()
print(stack)
```

```text:実行結果
[3, 4, 5, 6]
[3, 4, 5]
```

```python:サンプルコード
stack = ["a", "b", "c"]
stack.pop()
print(stack)
stack.append("d")
print(stack)
```

```text:実行結果
['a', 'b']
['a', 'b', 'd']
```

```python:サンプルコード
import queue
q = queue.LifoQueue()
x = [3, 4, 5]
for i in x:
    q.put(i)
while not q.empty():
    print(q.get())
```

```text:実行結果
5
4
3
```
