* [heapq.heapify()](https://docs.python.org/ja/3/library/heapq.html#heapq.heapify)： heap に変換する。

```python:サンプルコード
import heapq
hq = [3, 2, 5, 1, 4]
print("list:", hq)
heapq.heapify(hq)
print("hq:", hq)
```

```text:実行結果
list: [3, 2, 5, 1, 4]
hq: [1, 2, 5, 3, 4]
```
