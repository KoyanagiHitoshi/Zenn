* [round()](https://docs.python.org/ja/3/library/functions.html#round)：小数部を偶数に丸めた値を返す。

```python:サンプルコード
print("-2.5:", round(-2.5))
print("-1.5:", round(-1.5))
print("-0.5:", round(-0.5))
print(" 0.5:", round(0.5))
print(" 1.5:", round(1.5))
print(" 2.5:", round(2.5))
```

```text:実行結果
-2.5: -2
-1.5: -2
-0.5: 0
 0.5: 0
 1.5: 2
 2.5: 2
```
