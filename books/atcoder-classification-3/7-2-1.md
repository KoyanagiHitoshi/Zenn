* [calendar.isleap()](https://docs.python.org/ja/3/library/calendar.html#calendar.isleap)：閏年なら`True`を、そうでなければ`False`を返す。

```python:サンプルコード：sample_561.py
import calendar
print(calendar.isleap(2020))
```

```text:実行結果
True
```
