* [calendar.leapdays()](https://docs.python.org/ja/3/library/calendar.html#calendar.leapdays)：範囲指定された期間の閏年の回数を返す。

```python:サンプルコード
import calendar
print(calendar.leapdays(2020, 2025))
```

```text:実行結果
2
```
