* [sorted()](https://docs.python.org/ja/3/library/functions.html#sorted)：iterable の要素を並べ替えた新たなリストを返す。

昇順

```python:サンプルコード：sample_528.py
x = [2, 1, 3]
print(sorted(x))
```

```text:実行結果
[1, 2, 3]
```

降順

```python:サンプルコード：sample_529.py
x = [2, 1, 3]
print(sorted(x)[::-1])
```

```text:実行結果
[3, 2, 1]
```
