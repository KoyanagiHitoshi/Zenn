* [numpy.array()](https://numpy.org/devdocs/reference/generated/numpy.array.html#numpy.array)

```python:サンプルコード
import numpy
x = numpy.array([1, 2])
y = numpy.array([3, 4])
print(x+y)
print(x-y)
print(x*y)
```

```text:実行結果
[4 6]
[-2 -2]
[3 8]
```

```python:サンプルコード
import numpy
x = numpy.array([1, 2])
y = numpy.array([3, 4])
print((x*y).sum())
```

```text:実行結果
11
```

```python:サンプルコード
import numpy
x = numpy.array([1, 2])
y = numpy.array([3, 4])
print(numpy.dot(x, y))
```

```text:実行結果
11
```
