* [set()](https://docs.python.org/ja/3/library/stdtypes.html#set-types-set-frozenset)

```python:サンプルコード：sample_269.py
x = "abcabc"
print(set(x))
```

```text:実行結果
{'a', 'c', 'b'}
```

```python:サンプルコード：sample_270.py
x = "abcabc"
print(len(set(x)))
```

```text:実行結果
3
```
