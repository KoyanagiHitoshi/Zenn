* [str.join()](https://docs.python.org/ja/3/library/stdtypes.html#str.join)：要素の結合

```python:サンプルコード
x = ["a", "b", "c", "d", "e"]
print("".join(x))
```

```text:実行結果
abcde
```

```python:サンプルコード
x = ["a", "b", "c", "d", "e"]
print(",".join(x))
```

```text:実行結果
a,b,c,d,e
```

```python:サンプルコード
x = ["a", "b", "c", "d", "e"]
print(" ".join(x))
```

```text:実行結果
a b c d e
```
