* [数値型](https://docs.python.org/ja/3/library/stdtypes.html#numeric-types-int-float-complex)
* [complex](https://docs.python.org/ja/3/library/functions.html#complex)：複素数を返すか、文字列や数を複素数に変換する。

```python:サンプルコード
x = 1+2j
print(x)
print(type(x))
```

```text:実行結果
(1+2j)
<class 'complex'>
```

```python:サンプルコード
x = 3+4j
print(abs(x))
```

```text:実行結果
5.0
```
