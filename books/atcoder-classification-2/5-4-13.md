* [set()](https://docs.python.org/ja/3/library/stdtypes.html#set-types-set-frozenset)

```python:サンプルコード
x = [1, 2, 3, 2, 1, 3]
print(list(set(x)))
```

```text:実行結果
[1, 2, 3]
```

```python:サンプルコード
x = ["b", "a", "c", "a", "b"]
print(list(set(x)))
```

```text:実行結果
['b', 'c', 'a']
```

```python:サンプルコード
x = [[1, 0], [0, 0], [1, 1], [1, 0], [0, 1], [0, 0]]
print(list(map(list, set(map(tuple, x)))))
```

```text:実行結果
[[1, 0], [0, 1], [1, 1], [0, 0]]
```
