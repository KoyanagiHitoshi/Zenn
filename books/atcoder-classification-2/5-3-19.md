* [str.zfill()](https://docs.python.org/ja/3/library/stdtypes.html#str.zfill)：`0`で左詰めした文字列のコピーを返す。先頭が符号接頭辞（＋,－）だった場合、`0`は符号の後に挿入される。

```python:サンプルコード：sample_312.py
x = "12"
print(x.zfill(4))
```

```text:実行結果
0012
```

```python:サンプルコード：sample_313.py
x = "-12"
print(x.zfill(4))
```

```text:実行結果
-012
```
