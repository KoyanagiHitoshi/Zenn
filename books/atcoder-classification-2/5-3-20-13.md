```python:サンプルコード：sample_332.py
x = 10
print(f"{x = }")
print(f"{x = :b}")
print(f"{x*2 = }")
```

```text:実行結果
x = 10
x = 1010
x*2 = 20
```

```python:サンプルコード：sample_333.py
x = [1, 2, 3]
print(f"{x = }")
print(f"{x[0] = }")
```

```text:実行結果
x = [1, 2, 3]
x[0] = 1
```
