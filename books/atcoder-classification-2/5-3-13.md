* [str.replace()](https://docs.python.org/ja/3/library/stdtypes.html#str.replace)：文字列をコピーし、現れる部分文字列すべてを置換して返す。

```python:サンプルコード：sample_295.py
x = "abcabcabc"
print(x.replace("b", "p"))
```

```text:実行結果
apcapcapc
```

```python:サンプルコード：sample_296.py
x = "abcabcabc"
print(x.replace("b", "p", 2))
```

```text:実行結果
apcapcabc
```
