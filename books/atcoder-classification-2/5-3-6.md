title: "　　5.3.6 文字列のカウント"
---

* [str.count()](https://docs.python.org/ja/3/library/stdtypes.html#str.count)：重複せず出現する回数を返す。

```python:サンプルコード
x = "abbccc"
print(x.count("c"))
```

```text:実行結果
3
```

```python:サンプルコード
x = "abbccc"
print(x.count("d"))
```

```text:実行結果
0
```

```python:サンプルコード
x = "123435"
print(x.count("3"))
```

```text:実行結果
2
```
