* [共通のシーケンス演算](https://docs.python.org/ja/3/library/stdtypes.html#common-sequence-operations)

```python:サンプルコード：sample_293.py
x = "Hello World!"
y = x[:5]
print(y)
```

```text:実行結果
Hello
```

```python:サンプルコード：sample_294.py
x = "Hello World!"
y = x[:-1]
print(y)
```

```text:実行結果
Hello World
```
