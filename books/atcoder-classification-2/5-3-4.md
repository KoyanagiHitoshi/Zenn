* [共通のシーケンス演算](https://docs.python.org/ja/3/library/stdtypes.html#common-sequence-operations)

```python:サンプルコード
x = "abc"
print(x[0], x[1], x[2])
```

```text:実行結果
a b c
```

```python:サンプルコード
x = "abc"
print(x[-1], x[-2], x[-3])
```

```text:実行結果
c b a
```

```python:サンプルコード
x = "abc"
print(x[3])
```

```text:実行結果
IndexError: string index out of range
```
