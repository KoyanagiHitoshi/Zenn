* [float()](https://docs.python.org/ja/3/library/functions.html#float)：浮動小数点数を返す。

```python:サンプルコード：sample_247.py
print(float(10))
```

```text:実行結果
10.0
```

```python:サンプルコード：sample_248.py
print(10+0.0)
```

```text:実行結果
10.0
```

```python:サンプルコード：sample_249.py
print(1e3)
```

```text:実行結果
1000.0
```

```python:サンプルコード：sample_250.py
print(1e-3)
```

```text:実行結果
0.001
```
