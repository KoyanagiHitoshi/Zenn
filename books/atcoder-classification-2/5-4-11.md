* [list.count()](https://docs.python.org/ja/3/library/stdtypes.html#common-sequence-operations)：出現する回数

```python:サンプルコード
x = ["a", "b", "b", "c", "c", "c"]
print(x.count("c"))
```

```text:実行結果
3
```

```python:サンプルコード
x = ["a", "b", "b", "c", "c", "c"]
print(x.count("d"))
```

```text:実行結果
0
```

```python:サンプルコード
x = [1, 2, 3, 4, 3, 5]
print(x.count(3))
```

```text:実行結果
2
```
