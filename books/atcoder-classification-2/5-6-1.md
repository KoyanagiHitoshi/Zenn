```python:サンプルコード：sample_436.py
x = {"a": 1, "b": 2, "c": 3}
print(x["a"])
```

```text:実行結果
1
```

```python:サンプルコード：sample_437.py
x = {1: "a", 2: "b", 3: "c"}
print(x[2])
```

```text:実行結果
b
```

```python:サンプルコード：sample_438.py
x = dict(a=1, b=2, c=3)
print(x["a"])
```

```text:実行結果
1
```
