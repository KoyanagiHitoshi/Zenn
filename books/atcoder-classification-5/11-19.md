* [約数](https://ja.wikipedia.org/wiki/%E7%B4%84%E6%95%B0)

```python:サンプルコード
N = 100
divisors = [i for i in range(1, N+1) if N % i == 0]
print(divisors)
```

```text:実行結果
[1, 2, 4, 5, 10, 20, 25, 50, 100]
```

```python:サンプルコード
def divisors(num: int) -> list:
    ascend, descend = [], []
    i = 1
    while i*i <= num:
        if num % i == 0:
            ascend.append(i)
            if i != num//i:
                descend.append(num//i)
        i += 1
    return ascend + descend[::-1]


N = 100
print(divisors(N))
```

```text:実行結果
[1, 2, 4, 5, 10, 20, 25, 50, 100]
```
