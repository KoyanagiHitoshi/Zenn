* [素因数分解](https://ja.wikipedia.org/wiki/%E7%B4%A0%E5%9B%A0%E6%95%B0%E5%88%86%E8%A7%A3)

```python:サンプルコード
for num in range(2, 10):
    i = num
    Factoring = []
    flag = True
    while flag:
        for j in range(2, num+1):
            if i % j == 0:
                i = i/j
                if i == 1:
                    flag = False
                Factoring.append(j)
                break
    if len(Factoring) == 1:
        print(num, "is a prime")
    else:
        print(num, Factoring)
```

```text:実行結果
2 is a prime
3 is a prime
4 [2, 2]
5 is a prime
6 [2, 3]
7 is a prime
8 [2, 2, 2]
9 [3, 3]
```
